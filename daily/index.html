How to Play

You are given 16 different football-related categories arranged in a 4x4 grid. Each turn, a player name is shown — your task is to tap the cell that matches the player.

Correct: The cell fills with the player's name and turns green.

Wrong: The cell flashes red and you move to the next player.

Goal: Fill all 16 cells to get BINGO!

Wildcard: If the current player matches multiple empty cells, use the wildcard to fill them all at once.
